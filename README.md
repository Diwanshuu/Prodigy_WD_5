# 🌦 Weather App

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript**. The app fetches real-time weather data from the **OpenWeatherMap API** based on a user-entered city and displays the current weather conditions.

## 🚀 Features

* 🔍 Search weather by city name
* 🌡 Displays current temperature
* ☁️ Shows current weather condition
* 💧 Displays humidity
* 💨 Shows wind speed
* ⚠️ Handles invalid city names with error messages
* 📱 Fully responsive and modern UI
* ✨ Smooth animations using ScrollReveal

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API
